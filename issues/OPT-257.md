# OPT-257: Add stock logos next to ticker symbols using LogoKit API

Add company logos next to underlying ticker symbols in the positions list and detail view using LogoKit's Stock Logo API.

## Integration

Simple image URL: `https://img.logokit.com/ticker/{SYMBOL}`

* No API key required (free tier)
* 64×64 px on free tier, monogram fallback for unknown tickers
* CDN-delivered, <100ms globally

## Plan

* Add a small logo `<img>` next to each underlying symbol on the positions list page
* Add logo to the detail page header
* Consider adding to the ledger page position groups as well
* Use `loading="lazy"` for logos below the fold
* Handle the monogram fallback gracefully (it's automatic from LogoKit)
