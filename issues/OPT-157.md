# OPT-157: Add PostgreSQL maintenance to admin dashboard

Enrich the Database Health panel with maintenance operations and key performance indicators:

* **Enrich** `GET /api/admin/db-health`: Add dead tuples (per-table + total), cache hit ratio, index hit ratio, last autovacuum/autoanalyze timestamps
* **New** `POST /api/admin/db-maintenance/vacuum`: VACUUM ANALYZE with table whitelist validation, asyncio lock for concurrency, autocommit raw connection
* **Frontend**: `vacuumAnalyze()` and `vacuumAll()` methods in admin-app.js
* **Dashboard UI**: Performance metric cards (cache hit ratio, dead tuples), enhanced tables section with dead tuples column, last autovacuum column, and per-row VACUUM buttons
