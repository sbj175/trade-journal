# OPT-161: Full UI design refresh — Refined Terminal aesthetic

Complete visual redesign of OptionLedger with a "Refined Terminal" aesthetic — Bloomberg meets modern fintech.

## Scope

* **Global foundation**: New typography (Outfit + JetBrains Mono), refined color palette, CSS variables, animation utilities
* **Navigation**: Frosted glass nav bar, luminous active states, refined logo
* **Positions page**: Summary bar, table rows, expanded details, badges
* **Ledger page**: Status badges, filter bar, detail panels
* **Reports page**: Card/panel styling, chart containers
* **Risk page**: Greeks display, chart styling
* **Settings page**: Form elements, tabs, buttons

## Design Principles

* Monospace numbers for tabular data alignment
* Subtle depth through layered panels and shadows
* Smooth transitions (150-200ms) on all interactive elements
* LIVE indicator pulse animation
* Refined status badges and chips
* Frosted glass / backdrop-blur on elevated surfaces
