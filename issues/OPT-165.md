# OPT-165: Fix Positions page financial column data overlap

Dollar values in the Cost Basis, Net Liq, Realized, Open, Total, and % Rtn columns are overlapping on the Positions page. Large values like `-$138,963` and `$25,900.00` overflow the `w-24` (96px) column widths.

Fix:

* Widen financial columns from `w-24` to `w-[6.5rem]` (104px)
* Drop decimal places at $10K instead of $100K in `formatDollar`
* Add intermediate font size tier (`text-[13px]`) at $100K in `dollarSizeClass`
* Narrow Strategy/DTE/Days columns slightly to compensate
