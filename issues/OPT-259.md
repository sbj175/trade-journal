# OPT-259: Replace account icon with colored left border on position cards

The equities page and ledger page currently show an account indicator icon to the left of the ticker symbol. With stock logos now occupying that space ([OPT-257](https://linear.app/optionedge/issue/OPT-257/add-stock-logos-next-to-ticker-symbols-using-logokit-api), [OPT-258](https://linear.app/optionedge/issue/OPT-258/add-stock-logos-to-equities-positions-page)), we need a different approach for account identification.

## Plan

Replace the account icon with a thin colored left border (`border-l-4` or similar) on position cards/rows:

* Each account gets an assigned color
* The color bar runs the full height of the card/row
* Frees up the left-of-symbol space entirely for the stock logo
* Apply consistently across: equities positions page, ledger page, and options positions page (if multi-account indicators exist there)

## Design

* Assign colors per account (e.g., Roth IRA = blue, Individual = green, etc.)
* Store color assignments in account settings or derive from account type
* Subtle but distinct — the border should be visible without being loud

## Pages affected

* Equities positions (`pages/positions-equities/App.vue`)
* Ledger (`pages/ledger/App.vue`)
* Options positions (`pages/positions/App.vue`) — if applicable

## Depends on

Best done after [OPT-258](https://linear.app/optionedge/issue/OPT-258/add-stock-logos-to-equities-positions-page) (equities logos) so all logos are in place first.
