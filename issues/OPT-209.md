# OPT-209: Link exercised/assigned equity lots to their source option group

## Context

When options are exercised or assigned, the resulting equity lots have a `derived_from_lot_id` that links back to the source option lot. The `LotClosing` record also has a `resulting_lot_id` pointing forward to the equity lot. This bidirectional linkage exists in the database but isn't surfaced in the UI.

## Example

IBIT Roth IRA, Dec 31, 2025:

* Equity lot (id=80714, 600 shares, EXERCISE) → `derived_from_lot_id=80293`
* Option lot (id=80293, IBIT $47 Call) → LotClosing (type=EXERCISE) → `resulting_lot_id=80714`

## Idea

* On the Equities page, clicking an EXERCISE/ASSIGNMENT badge could navigate to the source option group on the Ledger
* On the Ledger, an exercised/assigned row could show a link icon that navigates to the resulting equity lot
* Helps users trace the full lifecycle: option strategy → exercise → share position
