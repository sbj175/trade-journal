# OPT-152: Positions page Tastytrade-style visual refresh (experiment)

## Summary

Experimental styling pass on the Positions page to bring the look and feel closer to Tastytrade's portfolio view. All changes were purely visual ā€” colors, fonts, font sizes, spacing ā€” with no functional or content changes (except adding "d" suffix to DTE values).

## Reference

Tastytrade's portfolio view (img_18) was used as the visual target. Key characteristics: very tight row density, solid colored backgrounds on price cells, P/L bar indicators, compact monospaced numbers, subtle row separators.

## Changes Made

### Tailwind Config (`frontend/tailwind.config.js`)

* `tv-muted` color: `#868c99` ā†’ `#d1d4dc` (matches `tv-text` for better readability)

### Row Density & Spacing (`frontend/src/pages/positions/App.vue`)

* Data row padding: `py-2` ā†’ `py-1`
* Header row padding: `py-2` ā†’ `py-1.5`
* Row dividers: `divide-tv-border` ā†’ `divide-tv-border/40` (more subtle)
* Main content padding unchanged (`p-2`)

### Font Sizes

* Column headers: `text-sm` ā†’ `text-xs`
* Symbol: `text-base` ā†’ `text-sm`
* Strategy label: `text-sm` ā†’ `text-xs`
* IVR: added `text-sm`
* DTE, Days: added `text-sm`
* All dollar columns (Cost Basis, Net Liq, Realized, Open P/L, Total, % Rtn): added `text-sm`
* Chevron icon: added `text-xs` and reduced opacity (`text-tv-muted/60`)

### Font Rendering

* All dollar columns: added `font-variant-numeric: tabular-nums` for proper monospaced number alignment

### Price Column

Restyled from `[price box with border] + % change text` to two flush colored boxes:

* Left box: `% change` with solid dark fill (`#1a3a1a` green / `#3a1a1a` red)
* Right box: `price` with slightly darker fill (`#132e13` green / `#2e1313` red)
* No borders, no rounded corners ā€” flush edge-to-edge like Tastytrade

### P/L Open Bar Indicator (see also OPT-151)

Replaced solid background fill with a directional bar graph:

* 4px gray track (`bg-tv-border/40`) spans full cell width ā€” always visible
* Green fill grows from **left edge** for profit
* Red fill grows from **right edge** for loss
* Width = `abs(openPnL / costBasis) * 100%`, capped at 100%
* Uses `relative overflow-hidden` container with absolute-positioned bar

### Tag Button

* Size: `w-4 h-4` ā†’ `w-5 h-5` (bigger click target)
* Shape: `rounded-full` ā†’ `rounded` (square with slight rounding)
* Icon: `+` text ā†’ `fa-tag` icon
* Border: more visible (`border-tv-border` instead of `border-tv-border/50`)
* Hover: blue tint with `hover:bg-tv-blue/10`

### DTE Column

* Added "d" suffix (e.g., "21d" instead of "21")
* Changed text color from `text-tv-text` to `text-tv-muted`

### Other Color Changes

* Days column: `text-tv-text` ā†’ `text-tv-muted`
* Net Liq: removed `font-medium` (was heavier than other dollar columns)
* Realized: removed `font-medium`
