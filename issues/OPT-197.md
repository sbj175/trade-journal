# OPT-197: Add TradingView chart widget with dynamic position watchlist

Embed a TradingView Advanced Chart widget on the Positions page with a dynamic watchlist populated from the user's open position underlyings.

## Implementation

1. Create a Vue 3 wrapper component that accepts a `symbols` prop (array of underlyings)
2. Render the TradingView Advanced Chart embed with `watchlist` parameter
3. Re-render when positions change (destroy/recreate widget)
4. Use dark theme to match our UI
5. Keep TradingView attribution visible (required for free use)

## Technical Details

* Free embeddable widget — no API key or data subscription needed
* Widget includes its own data feed (15-min delayed quotes)
* Shows underlying price charts only (not individual option contracts)
* Config example:

```javascript
{
  "symbol": "NASDAQ:AAPL",
  "watchlist": ["AAPL", "TSLA", "SPY"],
  "theme": "dark",
  "allow_symbol_change": true,
  "autosize": true
}
```

## Options

* `vue-tradingview-widgets` npm package provides Vue 3 components
* Or build a simple wrapper that manages the script tag lifecycle
* Could also use Lightweight Charts (Apache 2.0, \~45KB) for full control, but requires providing our own data

## Notes

* OptionIncome does exactly this — shows TradingView chart for the underlying alongside options position data
* Attribution (TradingView branding) must remain visible for free use
