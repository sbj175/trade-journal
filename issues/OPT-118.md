# OPT-118: Separate Connection & Import into dedicated setup page

Connection and Import Trades are onboarding/setup functions, not settings you tweak regularly. They should live on their own page (e.g. `/setup`) rather than being tabs in Settings.

**Current state:** Connection and Import Trades are the first two tabs in Settings alongside Privacy, Strategy Targets, and Roll Alerts.

**Proposed:**

* Create a new Setup/Onboarding page (`/setup`) with Connection and Import Trades
* Remove those two tabs from Settings — Settings keeps only Privacy, Strategy Targets, Roll Alerts
* Update the onboarding flow (`requireTastytrade()` redirect) to point to `/setup` instead of `/settings?tab=connection`
* Add a nav link for the new page
