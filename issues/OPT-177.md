# OPT-177: Add strategy and tag filters to Ledger, strategy filters to Reports

## Summary

Add filtering capabilities to the Ledger and Reports pages:

### Ledger

* **Strategy filter**: Filter groups by strategy label (e.g., Covered Call, Iron Condor, Short Put, etc.)
* **Tag filter**: Filter groups by user-assigned tags

### Reports

* **Strategy filter**: The Reports page already has a strategy breakdown, but needs an interactive filter to include/exclude specific strategies from the report data (currently has direction and type filters but no strategy picker)

### Notes

* Both pages already have the filter bar layout (two-row design with DateFilter on row 1, category filters on row 2)
* Strategy list can be populated from existing data (distinct `strategy_label` values from `position_groups`)
* Tag list can be populated from the existing `/api/tags` endpoint
