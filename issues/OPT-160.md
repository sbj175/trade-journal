# OPT-160: Sort ledger lot legs by strike ascending within same expiration

In the Ledger expanded view, open/partial lots with the same expiration are sorted by strike descending instead of ascending. This makes multi-leg positions harder to read (e.g. 150, 135, 145 instead of 135, 145, 150).
