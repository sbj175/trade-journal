# OPT-182: Add "Delta per qty" to Roll Analysis Greeks section

## Enhancement

Add a "Delta per qty" metric to the Roll Analysis panel alongside the existing Net Delta display.

## Details

* **Delta per qty** = Net Delta / GCD(all leg quantities)
* For a vertical spread with equal quantities (e.g., 5/5), GCD=5
* For a butterfly (e.g., 4/8/4), GCD=4
* For a ratio spread (e.g., 10/5), GCD=5
* This matches what Tastytrade calls "Delta per qty"
* Display next to or below the existing Net Delta value in the Greeks section of Roll Analysis
