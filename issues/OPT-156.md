# OPT-156: Expand roll analysis reward:risk to Iron Condors and other defined-risk strategies

## Context

The roll analysis box on the Positions page currently shows Remaining Reward, Remaining Risk, and Reward:Risk ratio for 4 spread strategies: Bull Put Spread, Bear Call Spread, Bull Call Spread, Bear Put Spread.

## Candidates for expansion

**Defined-risk (calculable max loss):**

* **Iron Condors** — 4 legs, defined risk via spread widths. Most natural next addition but detection/grouping across legs is more complex.
* **Iron Butterflies** — similar to Iron Condors
* **Covered Calls** — max loss involves stock cost basis minus call premium
* **Collars** — defined risk via protective put
* **Cash Secured Puts** — max loss = strike price minus premium received

**Non-defined risk (needs different approach):**

* Naked puts, naked calls, short strangles, short straddles
* Can show remaining reward (credit minus current P&L) but risk side has no natural cap
* Could use strategy target loss limit percentage as a proxy for max loss
* Needs a different UI treatment (e.g. "Remaining Reward: $X / Risk: undefined" or user-defined stop)

## Implementation notes

* Roll analysis lives in `getRollAnalysis()` in `frontend/src/pages/positions/App.vue`
* The `supportedStrategies` array on line \~851 controls which strategies get analysis
* Max profit/loss calculation (lines \~889-920) would need new branches per strategy type
* Backend `calculate_max_risk_reward()` in `src/services/report_service.py` already handles Iron Condors, Covered Calls, and naked options — could be referenced for the math
