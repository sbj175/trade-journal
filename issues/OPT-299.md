# OPT-299: Reorganize Settings sections under broader categories

The Settings page currently exposes a flat list of tabs that grew organically: Connection, Accounts, Import Trades, Privacy, Strategy Targets, Roll Alerts, Tags. OPT-298 adds an eighth, General. Eight flat tabs is past the point where users can predict where a given setting lives.

Propose grouping into broader categories. One plausible cut:

- **General** — colorblind palette, future display preferences (light/dark theme could also move here from the NavBar).
- **Data** — Connection, Accounts, Import Trades, Privacy.
- **Trading** — Strategy Targets, Roll Alerts.
- **Organization** — Tags.

UI choice: nested left-rail (sections → panels) vs. visually-distinct group headers in the existing flat list. Mobile dropdown grouping needs to work too.

## Notes

- OPT-298's General section is intentionally placed first in the flat list as a stepping stone — when the reorg happens, it stays at the top under its own category.
- No code changes in this issue yet — design + implementation when picked up.
