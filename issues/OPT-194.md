# OPT-194: Research Option Income

[https://www.optionincome.io](<https://www.optionincome.io>)

Claims "Real-time Greeks, live P&L, and everything you need to decide when to close, roll, or hold", but also claims to be "powered by SnapTrade or broker's direct connection".

Hacker news post: [https://news.ycombinator.com/item?id=46773157](<https://news.ycombinator.com/item?id=46773157>)

## Research Findings

### What It Does

OptionIncome is a SaaS options trading tracker and journal, targeting income-focused options sellers (covered calls, CSPs, credit spreads, wheel strategy). Read-only analytics on top of your existing broker.

### The Mystery: Where Do Their Greeks Come From?

**They calculate them.** SnapTrade gives them option contract details (strike, expiry, type) and a price for the underlying. They run Black-Scholes server-side to derive Greeks and back-solve for IV from the option's market price. Their "real-time" claim (5-10 second refresh) is just **polling** SnapTrade's holdings endpoint, not true streaming.

### Supported Brokers (6)

Robinhood, Charles Schwab, Fidelity, E\*Trade, Tastytrade, Webull. Coming soon: IBKR, Moomoo. All via SnapTrade.

### Pricing

* Pro Monthly: $19/month
* Pro Annual: \~$15/month
* Lifetime: $399 one-time
* 14-day free trial

### Feature Comparison

|  | OptionIncome | OptionLedger |
| -- | -- | -- |
| **Brokers** | 6 (via SnapTrade) | Tastytrade only |
| **Quotes** | Polling 5-10s | True WebSocket streaming |
| **Greeks** | Self-calculated | Self-calculated |
| **Chain analysis** | Basic roll/wheel linking | Deep chain + lot-level tracking |
| **Data privacy** | Cloud (their servers) | 100% local or self-hosted |
| **Pricing** | $19/mo | Free |
| **Charts** | TradingView embedded | Not included |
| **AI analytics** | Yes | No |

**Where they're ahead**: Multi-broker, TradingView charts, wheel strategy tracking, zero-setup cloud SaaS.

**Where we're ahead**: Real-time streaming, deeper chain/lot analysis, data privacy, free, richer direct API data.

**Notable**: Very small online footprint — no Reddit, no reviews, no Product Hunt. Very new or very small user base.
