# OPT-195: Research market data providers for broker-agnostic quotes

Per Tastytrade TOS, we can only serve TT quotes to TT customers. If we add other brokers (via SnapTrade or direct integrations), we need an independent market data source for live quotes and Greeks.

## Candidates to Evaluate

* **Tradier** — free tier, real-time quotes + Greeks (via ORATS), streaming API
* [**Polygon.io**](<http://Polygon.io>) — real-time options data, WebSocket streaming, starts at $199/mo for options
* **CBOE LiveVol / DataShop** — institutional-grade, expensive
* **OPRA feed** — source of all US options data, very expensive

## Key Requirements

* Real-time or near-real-time option quotes (bid/ask/last)
* Greeks (delta, gamma, theta, vega) or enough data to calculate them
* WebSocket or streaming support (polling is acceptable if fast enough)
* Reasonable cost that works with a per-user SaaS pricing model
* Options chain data (strikes, expirations)

## Architecture Considerations

* Could use Tastytrade WebSocket for TT customers + separate provider for others (hybrid)
* Or standardize on one provider for all users (simpler but loses TT streaming advantage)
* Need to evaluate latency, reliability, and data quality of each option
