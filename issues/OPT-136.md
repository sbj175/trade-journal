# OPT-136: Add PostgreSQL database health panel to admin dashboard

Add a "Database Health" section to the admin dashboard showing PostgreSQL-specific metrics:

* DB connectivity status and query latency
* Database size
* Schema version (from alembic_version)
* Connection pool stats (pool_size, checked_out, checked_in, overflow)
* Table sizes and row counts (sorted by size desc)

New endpoint: `GET /api/admin/db-health`
Frontend: Alpine.js state + panel in dashboard template
