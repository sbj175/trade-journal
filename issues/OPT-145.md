# OPT-145: Move Reprocess Chains from Settings to Admin Dashboard

Move the "Reprocess Chains" functionality from the user-facing Settings page to the admin dashboard as a per-user action.

**Changes:**

* Add `POST /api/admin/users/{user_id}/reprocess-chains` endpoint
* Add "Reprocess" button to Users table actions in admin dashboard
* Add `reprocessChains(user)` JS handler in admin-app.js
* Remove Reprocess Chains block from Settings page
* Remove `reprocessChains()` and `reprocessing` state from settings-app.js
* Remove `POST /api/reprocess-chains` endpoint from [sync.py](<http://sync.py>)
