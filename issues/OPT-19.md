# OPT-19: Chain link on Positions page should only open that chain

Clicking the chain link next to SLV for a single position (say Roth IRA) should link only to the Roth IRA SLV chain. Instead, it is showing all SLV chains. I think we just need to filter the account in addition to the symbol when clicking a chain.
