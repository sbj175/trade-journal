# OPT-226: Key Dashboards

If OptionLedger delivers just a few high-signal dashboards, traders will immediately feel it’s better than a spreadsheet or generic journal. The goal isn’t more charts—it’s answering the questions traders subconsciously ask every week.

Here’s the minimum professional-grade analytics stack I’d recommend building first (in priority order) 📊

\---

Dashboard 1 — Strategy Performance (the centerpiece)

Question it answers:

\> Which strategy is actually making me money?

Show:

Strategy	Trades	Win Rate	Avg Win	Avg Loss	Expectancy	Net P&L

Expectancy is the key metric:

Expectancy =

(win rate × avg win) − (loss rate × avg loss)

This instantly tells traders:

iron condors profitable?

vertical spreads leaking?

calendars outperforming?

Most journals don’t compute expectancy correctly.

That’s your edge.

\---

Dashboard 2 — Profit by Symbol

Question it answers:

\> Which tickers should I trade more?

Show:

Symbol	Trades	Win Rate	Net P&L	Avg Return

Common discovery:

\> “SPX prints money. TSLA drains it.”

This alone changes behavior.

This is where OptionLedger can become special.

\> Are my adjustments helping or hurting?

Retail traders almost never measure this.
