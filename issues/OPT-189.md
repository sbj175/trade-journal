# OPT-189: Custom domain: app.optionledger.app → Railway

Set up custom domain to hide the Railway URL.

## Steps

1. Add `app.optionledger.app` as a custom domain in Railway (Settings → Networking)
2. Add CNAME record in DNS: `app.optionledger.app → web-production-a042b.up.railway.app`
3. Update `TASTYTRADE_REDIRECT_URI` to `https://app.optionledger.app/auth/tastytrade/callback`
4. Update Supabase Auth Site URL to `https://app.optionledger.app`
5. Update Supabase Auth Redirect URLs to include `https://app.optionledger.app/**`
6. Register the new redirect URI in Tastytrade OAuth app settings

## Notes

* Railway handles SSL automatically
* Root domain (`optionledger.app`) stays for the landing page (hosted separately)
* The Railway-generated URL (`web-production-a042b.up.railway.app`) will still work but won't be user-facing
