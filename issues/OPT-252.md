# OPT-252: Landing: Google Search Console verification meta tag

Add the Google Search Console verification meta tag to `landing/index.html` so the property can be verified and the sitemap submitted.

Tag provided by GSC:

```
<meta name="google-site-verification" content="VmqcI_XCt7-f3xr_CXVDdjLfGTPulTY-q6uKLB9KTMs" />
```

After deploy, click Verify in GSC, then submit `sitemap.xml` under the Sitemaps tab.
