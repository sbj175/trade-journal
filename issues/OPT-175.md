# OPT-175: Build shared DateFilter component (TT-style calendar picker)

Build a reusable DateFilter Vue component inspired by Tastytrade's date range picker. Replace the existing time filter dropdowns on Ledger and Reports pages.

**Features:**

* Compact dormant bar showing date range + calendar icon
* Expanded dropdown with preset sidebar (Today, Yesterday, 7/14/30/60/120 days, YTD, Custom)
* Single-month calendar with range highlighting and month/year navigation
* Custom mode: clickable From/To dates with "Select date" state
* Uses our Deep Finance color palette
* Emits `update` event with `{ from, to }` dates
* Persists selection to localStorage
