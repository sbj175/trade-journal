# OPT-77: Positions page too crowded — add column visibility or display modes

The Positions page has accumulated many columns (Symbol, IVR, Price, Strategy, DTE, Days, Cost Basis, Net Liq, Realized, Open, Total, % Rtn, % Max) and is becoming hard to scan. Consider a mode toggle or column visibility controls to reduce density.

Possible approach: a display mode toggle similar to Wingman's "Cost Basis" vs "P/L" modes — one focused on cost basis / per-unit values, the other on P/L metrics.
