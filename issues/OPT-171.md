# OPT-171: Add setting to toggle ledger merge suggestions on/off

Add a toggle on the Settings page to enable/disable merge suggestions on the Ledger page. When disabled, the `GET /api/ledger/suggestions` call should be skipped entirely.

Store the preference in localStorage (single-user) or user settings table (multi-user).

## Comments

### 2026-03-30 — Steve Johnson

Closing as irrelevant. The backend `/api/ledger/suggestions` endpoint exists but the frontend Ledger page no longer calls it — the merge suggestions feature is unused. A settings toggle for a feature that isn't surfaced in the UI doesn't make sense. If merge suggestions are re-introduced in the future, a toggle can be added at that point.
